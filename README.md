# 🎓 Exam Guard Monitoring System

A web-based Exam Guard Monitoring System developed as an academic project to help conduct online examinations with basic monitoring and exam management features.

---

## 📌 Project Overview

The Exam Guard Monitoring System is built using Flask and provides an online platform where students can log in, access exam materials, take examinations, and receive results. The project aims to simplify the online examination process through a user-friendly interface.

---

## 🚀 Features

- 👨‍🎓 Student Login
- 👨‍💼 Admin Dashboard
- 📝 Online Examination
- 📊 Result Generation
- 📚 Study Material (PDFs)
- 🎥 Educational Videos
- 🌐 Responsive User Interface

---

## 🛠 Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/Riya04-web/Exam-Guard-Monitoring.git
```

Move into the project folder

```bash
cd Exam-Guard-Monitoring
```

Install dependencies

```bash
pip install flask
```

Run the application

```bash
python app.py
```
